# Covid-19-CLI

## Project setup
```
npm install -g .
```

### Run Project From any CMD
```
covid-19
```

### Check Cases of a Country(like for Pakistan its country code is 'PK')
```
covid-19 -n PK
```

### For USA
```
covid-19 -n US
```
